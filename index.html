Welkom op mijn portfolio

Hallo! Mijn naam is Yunus Özcan, en dit is mijn portfolio. Hier kunt u mijn werk en projecten bekijken die ik tijdens mijn studie heb voltooid. Elk project is een weerspiegeling van mijn groei en vaardigheden op het gebied van webontwikkeling en IT-oplossingen. Blader gerust door mijn werk, en aarzel niet om contact op te nemen als u vragen heeft of wilt samenwerken!

Project 1: Website Ontwerp

Een eenvoudige website gebouwd met HTML en CSS. Het doel was om een strakke en responsieve interface te creëren. Het project bevat verschillende pagina's, waaronder een homepage en een contactpagina. Ik heb gebruik gemaakt van moderne designprincipes zoals typografie en kleurenschema's. Dit project benadrukt mijn vaardigheden in front-end webontwikkeling.

Project 2: Netwerk Configuratie

Dit project ging over het opzetten van een netwerk voor een klein bedrijf. Het omvatte het configureren van routers, switches en firewalls. Ik heb gewerkt met verschillende netwerktechnologieën zoals VLAN’s en IP-adressering. Het doel was om een stabiel en veilig netwerk te garanderen. Dit project geeft inzicht in mijn netwerkbeheer vaardigheden.

Project 3: IT Support Tool

Ik ontwikkelde een Python-script om veelvoorkomende netwerkproblemen op te lossen. Het script helpt bij het controleren van IP-connectiviteit en het uitvoeren van diagnostische tests. Het was bedoeld voor gebruik door IT-medewerkers in kleine bedrijven. Het vereenvoudigt en versnelt de probleemoplossing. Dit project toont mijn kennis van automatisering en scripting.

Project 4: Databasebeheer

Dit project richtte zich op het ontwerpen en beheren van een eenvoudige database voor klantgegevens. Ik gebruikte MySQL voor de database en PHP voor de interface. De focus lag op databasetabellen, relaties en eenvoudige query's. Het doel was om de gegevens efficiënt op te slaan en te beheren. Dit project toont mijn begrip van databases en server-side scripting.

Project 5: Cloud Implementatie

In dit project implementeerde ik een cloudopslag oplossing voor veilige bestandshosting en delen. Ik gebruikte AWS om een S3-bucket te creëren en bestanden te beheren. Het doel was om de toegankelijkheid en veiligheid van bestanden voor kleine bedrijven te verbeteren. Dit project illustreert mijn kennis van cloud computing en best practices voor gegevensbeveiliging.

Project 6: Beveiligingsaudit

Dit project betrof een beveiligingsaudit voor een bedrijfsnetwerk. Ik heb verschillende kwetsbaarheden geïdentificeerd en aanbevelingen gedaan voor verbeteringen. Het auditrapport omvatte firewalls, toegangspunten en netwerkinstellingen. Dit project toont mijn vaardigheden in netwerkbeveiliging en risicobeheer.
